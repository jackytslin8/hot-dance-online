# 熱舞 Online 懷舊遊戲 - 開發規格

## 核心玩法
1. **箭頭輸入系統**: 空心方向光標 → 輸入方向鍵 → Space 判定
2. **方向**: 四方向 (↑↓←→) + 八方向 (加入 ↖↗↙↘)
3. **判定等級**: Perfect > Great > Cool > Bad > Miss
4. **Combo 系統**: 連續成功增加 Combo，倍率提升
5. **Fever/爆氣**: 集氣條滿後爆氣，分數加倍
6. **長按 (Hold)**: 持續按住直到尾端
7. **連打 (Mash)**: 快速連擊指定次數
8. **速度設定**: x1, x2, x4
9. **反鍵 (Reverse)**: 箭頭變紅，方向相反
10. **隱藏模式**: Hidden / Sudden / 神隱

## 遊戲模式
- **單人練習**: 自己打歌
- **PK 模式**: 跟 AI 或其他玩家比
- **情侶模式**: 兩人各打一半

## 畫面特色
- 七彩霓虹介面
- 閃爍 Combo 字體
- 大量 glow 特效
- 韓系 UI 泡泡字體
- 角色自動跳舞

## 技術棧

- Web Audio API (節奏同步)
